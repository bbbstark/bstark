运算符

递增递减运算

  • 只能用于变量
运算符 名称 描述
++ x$ 预递增 x 加 1，然后返回 x
$x ++ 后递增 返回 x，然后 x 加 1
– x$ 预递减 x 减 1，然后返回 x
$x – 后递减 返回 x，然后 x 减 1
运算符 名称 等同于 描述
== 等于 $x==$y 如果$x等于$y,返回true
=== 全等于（完全相同） $x===$y 如果$x等于$y,且类型相同返回true
!= 不等于 $x!=$y 如果$x不等于$y,返回true
<> 不等于 $x<>$y 如果$x不等于$y,返回true
!== 不全等（属性和值都不等于） $x!==$y 如果$x不等于$y,且类型不相同返回true
> 大于 $x>$y 如果$x大于$y,返回true
< 小于 $x<$y 如果$x小于$y,返回true
>= 大于或等于 $x>=$y 如果$x大于等于$y,返回true
<= 小于或等于 $x<=$y 如果$x小于等于$y,返回true
and,&& 与 $x and$y 如果$x和$y都为true，则返回true
or,|| 或 $x or $y 如果$x和$y至少1个为true，则返回true
not,! 非 !$x 如果$x不为true，则返回true
xor 异或 $x xor $y 如果$x和$y有且仅有1个为true，则返回true

数组的创建和读取

创建数组的基本方法是应用array()函数

  • 索引数组 key为连续的整数，默认从下标0开始。
  • 关联数组 key为指定的值，通常为字符串
  • 多维数组

数组的基本操作

  • 通过赋值增加数组元素
  • 通过使用array_push()函数追加数组元素(入栈)
  • 通过使用array_pop()函数删除最后一个数组元素出入栈)
  • 数组的排序
排序函数 作用 属性
sort() 以升序对数组排序 基本排序
rsort() 以降序对数组排序 基本排序
ksort() 根据键值，以升序对关联数组进行排序 键值排序
krsort() 根据键值，以降序对关联数组进行排序 键值排序
asort() 根据值，以升序对关联数组进行排序 值排序
arsort() 根据值，以降序对关联数组进行排序 值排序

判断语句

  • if 语句 - 在条件成立时执行代码
  • if…else 语句 - 在条件成立时执行一块代码，条件不成立时执行另一块代码
  • if…elseif….else 语句 - 在若干条件之一成立时执行一个代码块
  • switch case 语句 - 在若干条件之一成立时执行一个代码块

循环语句

  • while - 只要指定的条件成立，则循环执行代码块
  • do…while - 首先执行一次代码块，然后在指定的条件成立时重复这个循环
  • for - 循环执行代码块指定的次数
  • foreach($array as $value) - 根据数组中每个元素来循环代码块 ps：js里用的是 for in

类

  • 关键字class开头

  • 类是抽象的对象，对象是具体化的类。

  • 可理解为类是带参数的对象，是一个空白的对象模板。

  • 人类是一个类，比尔盖茨则是一个对象。

  • 类有属性(变量) 和方法(函数)

  • 代码重复使用、灵活性强，减少代码重复臃肿，提高执行效率。

php文件读取

操作步骤 函数名称 作用
打开文件1 fopen() 打开文件
文件读取2 fread() 读取文件指定长度的字符串
文件读取3 fgets() 读取文件一行内容
文件读取4 feof() 检测是否已经到达文件末尾
文件写入5 fwrite() 写入数据
关闭文件6 fclose() 关闭文件

  • 打开文件常用函数

    fopen()参数 说明
    r 以只读的方式打开，如果文件不存在将出错
    w 以写入的方式打开，将文件指针指向文件头部，并删除文件内容，如果文件不存在则创建
    a 以追加写入的方式打开，将文件指针指向文件末尾，如果文件不存在则创建
    r+ 以读写的方式(先读后写)打开，将文件指针指向文件头部
    w+ 以读写的方式打开，将文件指针指向文件头部，并删除文件内容
    a+ 以追加读写的方式打开，将文件指针指向文件末尾
    x 以只写的方式创建并打开文件，并将文件指针指向文件头，如果文件存在，就会打开失败
    x+ 以读写的方式创建并打开文件，并将文件指针指向文件头。如果指定文件存在，就会打开失败
    b 以二进制模式打开，可与r\w\a合用

  • 读取文件常用函数

函数 说明
