Nmap 常用命令

格式：
nmap [ <扫描类型> …] [ <选项> ] { <扫描目标说明> }

  • -sP/-sn 使用ping扫描,查看主机存活,不进行端口扫描
  • -P 指定端口扫描
  • sT TCP全连接扫描,建立完整的三次握手过程
  • sS SYN半连接扫描,只进行三次握手的前两次
  • -sV 显示详细的版本信息
  • -O 显示操作系统版本
  • Pn 不对主机进行ping检测,直接扫描该主机
